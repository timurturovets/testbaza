# TestBaza

Backend и frontend проекта TestBaza - платформы для создания, публикации и прохождения пользовательских тестов.

## Функционал

Создание и редактирование тестов:
- создание теста с описанием, приватным доступом по ссылке и загрузкой изображений
- добавление/редактирование вопросов с поддержкой двух типов ответов:
- ввод текста пользователем
- выбор из нескольких вариантов
- настройка ограничений по времени и количеству попыток
- публикация теста с расширенной серверной валидацией структуры вопросов и ответов

Прохождение тестов:
- прохождение опубликованных тестов (публично или по приватной ссылке)
- сохранение ответов пользователя в процессе попытки
- завершение попытки с созданием данных для проверки
- автоматическая проверка или ручная проверка автором теста

Личный кабинет и модерация:
- профиль пользователя (данные аккаунта, смена пароля, список своих тестов)
- статистика по прохождениям и детальная аналитика попыток
- оценка (rating) пройденных тестов
- админ-панель для удаления тестов и пользователей (по `ADMIN_USERNAME`)

## Стек технологий

- C# / ASP.NET Core MVC (.NET 6)
- ASP.NET Core Identity
- Entity Framework Core + SQL Server
- React 17 (UMD) + Babel Standalone
- Bootstrap 5

Проект создавался в учебных целях.

Некоторые части проекта нуждаются в рефакторинге.
- унифицировать REST-маршруты 
- усилить слой валидации и стандартизацию формата ошибок API
- разделить ответственность контроллеров и вынести часть бизнес-логики в сервисный слой
- улучшить middleware для обновления БД и обработку edge-case сценариев конкурентного доступа
